Intuitive Design

CaDoodle CAD features an intuitive interface that makes creating complex designs accessible to everyone, from beginners to professionals.

Powerful Plugins

Extend the CaDoodle workflow with file type support for Blender, FreeCAD, Inkscape, OpenSCAD, BolwerStudio and STL.

Local Application

CaDoodle is a local application that uses the resources of the computer running it. Models are stored on your own computer.

Tinkercad Alternative

CaDoodle offers a familiar drag-and-drop workflow for Tinkercad users, but works completely offline with no internet required. Perfect for schools and education.
